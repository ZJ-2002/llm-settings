# 跨学科联想引擎 v2.5.0

## 功能定位

本模块突破单一疾病领域的局限，主动检索**跨学科相似逻辑**，借鉴其他领域的研究成果和机制理解。

## 核心概念: 科学类比

结构: 源领域 → 相似机制 → 目标领域应用

## 联想类型

### 1. 机制类比
- 肿瘤微环境 → 椎间盘退变
- 骨关节炎 → 椎间盘退变

### 2. 方法学类比
- 液体活检 → 脊柱感染
- 风险评估模型 → 脊柱手术

### 3. 技术类比
- 影像引导消融 → 椎间盘
- 机器人辅助 → 脊柱手术

## 质量控制

使用 biological-plausibility-gate.md 进行生物学可行性验证。

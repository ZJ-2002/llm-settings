# 证据综合与知识图谱子技能 v2.4.0

## 概述

从已筛选的文献中系统提取观点，识别共识、争议和空白点，构建结构化知识图谱。

## 核心功能

### 1. Evidence Audit Trail

强制附加在每条观点后的完整学术批注。

### 2. Conflict Resolver

深度分析争议来源。

### 3. ERS 证据稳健性评分

五维度量化评估系统。

### 4. Adversarial Synthesis

AI扮演审稿人提出反直觉病例。

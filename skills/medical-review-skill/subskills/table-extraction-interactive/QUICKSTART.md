# 交互式表格提取 - 快速开始

## 使用方式

### 方式1：自然语言触发

```
你: 提取这篇文献的表格

AI: 开始提取表格...
    ✅ 提取完成！
    检测到 3 个表格
```

### 方式2：命令触发

```
/medical-review-skill extract-tables
```

## 支持的回复

| 意图 | 可以说的 |
|------|----------|
| 确认 | "是" / "正确" / "对" |
| 否定 | "否" / "错误" / "不对" |
| 修正 | "改为 48.5" / "应该是 48.5" |
| 查看原文 | "查看原文" / "看PDF" |
| 全部确认 | "全部正确" / "都对" |

## 工作流程

```
提取表格 → 显示摘要 → 分流处理 → 人工复核 → 数据入库
```

**无需命令行，全程对话式操作**

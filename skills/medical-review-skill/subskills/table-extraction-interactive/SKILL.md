# 交互式表格提取子技能 v1.0.0

## 概述

零命令行的表格提取体验：
- AI自动检测和提取PDF中的表格
- 智能分流（A/B/C级置信度）
- 对话式人工复核界面

## 触发方式

- `/medical-review-skill extract-tables`
- 自然语言："提取这篇文献的表格"

## 分流策略

| 级别 | 置信度 | 处理方式 |
|------|--------|----------|
| A级 | >=90% | 自动入库 |
| B级 | 70-90% | 批量确认 |
| C级 | <70% | 逐项核对 |

## 交互命令

- `提取表格`: 开始提取
- `是`/`否`: 确认/否定
- `改为 [值]`: 修正数据
- `查看原文`: 显示PDF截图
- `全部正确`: 一键确认

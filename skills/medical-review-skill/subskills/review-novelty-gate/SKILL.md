# Review Novelty Gate - 综述价值评估门槛

## 核心目标

通过结构化评估，决定综述项目是否值得继续投入资源。

## 决策类型

- **Go**: 继续推进
- **No-Go**: 终止项目
- **Revise-and-Proceed**: 修订后继续

## 评估步骤

1. 核心综述问题定义
2. 知识缺口量化（总分≥14强烈推荐）
3. 与现有综述对比
4. 证据缺口分析
5. 贡献分类（A/B/C/D）
6. 反事实分析

## A/B/C/D 分类

- **A类**: 强烈推荐（总分≥14）
- **B类**: 推荐撰写（总分10-13）
- **C类**: 有条件推荐（总分7-9）
- **D类**: 不推荐撰写（总分<7）

## 可行性熔断

- 近5年高质量RCT+队列≥3篇
- 预计全文可获取率≥60%

# 视觉逻辑引擎 (Visual Logic Engine) v2.5.1

## 概述

将医学综述中的复杂描述转化为可执行的视觉化指令。

## 输出格式

1. **Mermaid流程图**: 机制通路、临床路径
2. **BioRender指令**: 专业级模式图
3. **图表-数据锚点**: 与正文关联

## 工作流程

1. 视觉要素提取（实体、关系、层级）
2. 逻辑结构建模（因果链、反馈环路）
3. 多格式输出

## v2.5.1 BioRender自动化升级

直接输出可解析的图层描述规范：
- 画布设置
- 图层描述（背景层、解剖层、病理层等）
- 专科专用模板（LDH手术示意图、分子机制图）

## 图表类型

- 疾病机制模式图
- 临床诊疗路径图
- 分子信号通路图
- 时间轴/疾病进展图

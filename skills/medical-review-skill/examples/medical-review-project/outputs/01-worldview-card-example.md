## 综述故事线

```
2型糖尿病患病率持续攀升，运动干预是管理基石(背景) →
现有指南推荐运动但缺乏个体化处方细节(现状) →

适用于病情稳定的成人T2DM患者(边界)
```

## 核心学术洞见

1. **知识缺口识别**
   - 现有综述多聚焦于"运动vs不运动"，缺乏不同类型运动的头对头比较
   - 剂量-效应关系（强度、频率、时长）的证据分散

2. **预期贡献**
   - 本综述将整合近5年RCT和系统评价，明确：
     - 有氧 vs 抗阻 vs 混合运动的相对优劣
     - 最佳运动剂量参数
     - 特殊人群（老年、肥胖）的运动处方

3. **学术价值声明**
   - 为内分泌科临床医生提供基于最新证据的运动处方决策参考
   - 预计纳入50-80篇高质量文献，包含15-20篇RCT

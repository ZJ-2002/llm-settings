### 2.1 2型糖尿病患病率与发病率

- 久坐行为与代谢风险

### 2.3 危险因素与高危人群
- 可干预因素：肥胖、饮食、久坐
- 不可干预因素：遗传、年龄
- 高危人群特征：前期糖尿病、肥胖、家族史

**预计字数**: 1000-1200字

### 3.1 运动改善血糖的生理机制
- **骨骼肌机制**：
  - GLUT4转位增加
  - 胰岛素敏感性改善
  - 肌糖原合成增加
- **肝脏机制**：
  - 肝糖输出减少
  - 糖异生抑制
- **脂肪组织机制**：
  - 脂肪氧化增加
  - 炎症因子减少

### 3.2 不同类型运动的机制差异
- **有氧运动**：
  - 主要改善外周胰岛素敏感性
  - 能量消耗大，减脂效果好
- **抗阻运动**：
  - 增加肌肉量，提升基础代谢
  - 改善肌肉内脂质代谢
- **混合运动**：
  - 综合获益，协同效应

### 3.3 运动对并发症的保护机制
- 心血管保护
- 神经保护

## 第五章：管理 (Management)
**预计字数**: 1500-2000字（重点章节）

### 5.1 运动类型比较
- **有氧运动**：
  - 形式：快走、慢跑、游泳、骑车
  - 证据：降低HbA1c 0.5-0.7%
  - 最佳参数：中等强度，150分钟/周
- **抗阻运动**：
  - 形式：器械训练、弹力带、自重训练
  - 证据：降低HbA1c 0.3-0.5%，增加肌肉量
  - 最佳参数：2-3次/周，8-10个动作
- **混合运动**：
  - 证据：降低HbA1c 0.8-1.0%，效果最优
  - 推荐方案：有氧+抗阻组合

### 5.2 运动剂量-效应关系
- **强度**：中等 vs 高强度比较
- **频率**：每周次数与效果
- **时长**：每次持续时间
- **总运动量**：每周MET-minutes目标

### 5.3 特殊人群的运动处方
- **老年患者**：低强度、平衡训练、防跌倒
- **肥胖患者**：关注关节保护、循序渐进
- **合并心血管疾病**：心脏康复模式
- **糖尿病前期**：预防性运动方案

### 5.4 运动安全性
- 低血糖风险与预防
- 运动相关不良反应
- 监测建议

### 5.5 提高依从性的策略
- 行为改变理论应用
- 远程监测与支持
- 社区资源利用

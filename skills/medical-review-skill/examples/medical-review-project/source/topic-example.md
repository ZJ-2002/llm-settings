# 综述选题描述

## 标题
2型糖尿病运动干预研究进展

## 研究背景
2型糖尿病是全球最常见的慢性代谢性疾病之一，患病率持续攀升。运动干预作为糖尿病管理的基石之一，在血糖控制、并发症预防和生活质量改善方面具有重要作用。然而，不同类型、强度、时长的运动干预效果存在差异，最佳运动处方尚无统一共识。

## 研究目的
系统梳理近年来关于2型糖尿病运动干预的研究进展，包括：
1. 不同运动类型（有氧运动、抗阻运动、混合运动）的效果比较
2. 运动干预的剂量效应关系
3. 运动干预的作用机制
4. 特殊人群的运动处方建议
